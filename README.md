# 法律助手技能集

适用于 Claude Code、Codex 及其他平台的法律工作 AI 助手技能。

> **免责声明**：本仓库仅提供软件工具，不构成法律建议。

## 技能列表

### contract-review
合同审查技能，通过添加评论式问题标注来辅助合同审查，不修改原文。功能包括：

- 三层审查清单（基础/商业/法律）
- 结构化评论（问题类型、风险原因、修订建议）
- 风险等级通过审核者名称编码
- 生成合同摘要、综合意见和 Mermaid 业务流程图

### law-to-markdown
将法条/规范文件（`.txt`/`.docx`/`.pdf`）转换为 Markdown 格式。支持多种输入格式，采用三阶段处理流程（转换 → 格式化 → 校验），自动识别中国法律文本结构，生成规范的 Markdown 文件和详细的审核报告。

**标题层级对应：**
| 层级 | Markdown |
|------|----------|
| 法律名称 | `#` |
| 编/分编 | `##` |
| 章 | `###` |
| 节 | `####` |
| 条 | `#####` |
| 款/项/目 | -（无标题）|

### ad-compliance-review
广告合规审核技能，用于审核广告素材是否符合中国广告法及相关法规。采用四阶段流水线审核（定性分类 → 通用红线 → 行业专项 → 生成报告），涵盖《广告法》《反不正当竞争法》及房地产、食品、医疗、药品、互联网等 20+ 行业专项法规。

### food-label-review
预包装食品标签合规审核技能，用于审核食品标签是否符合 GB 7718（预包装食品标签通则）和 GB 28050（预包装食品营养标签通则）。支持 2011 版和 2025 版标准，涵盖一般标签审核（食品名称、配料表、净含量等）和营养标签审核（营养成分表、营养声称、NRV% 等）。
