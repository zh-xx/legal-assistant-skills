# NRV 与允许误差 (GB 28050-2011)

## 检查清单

### NRV 计算

- [ ] NRV% = (食品中营养素含量 / 该营养素 NRV) × 100%
- [ ] NRV% 修约间隔为 **1**（如 1%、5%、16%）
- [ ] 未规定 NRV 的营养成分**仅需标示含量**，不做 NRV% 计算

### "0" 界限值

- [ ] 每 100g 或 100mL 中营养成分含量 ≤ "0" 界限值时，应标示为 **"0"**
- [ ] 用"份"标示时，也须同时符合每 100g/100mL 的 "0" 界限值规定

### 允许误差（表 2）

| 类别 | 允许误差 |
|------|---------|
| 蛋白质、多不饱和/单不饱和脂肪(酸)、碳水化合物、糖(仅限乳糖)、膳食纤维、维生素(不含 VD/VA)、矿物质(不含钠)、强化的其他营养成分 | **≥80% 标示值** |
| 能量、脂肪、饱和脂肪(酸)、反式脂肪(酸)、胆固醇、钠、糖(除外乳糖) | **≤120% 标示值** |
| **维生素 A、维生素 D** | **80%～180% 标示值** |

### 审查要点

- [ ] 各营养素 NRV% 计算是否正确
- [ ] 含量标示值是否为修约间隔的整数倍
- [ ] 达到 "0" 界限值的营养素是否标示为 "0"
- [ ] 允许误差范围是否符合表 2 规定
