# 生产者与经营者信息审核检查项（GB 7718-2025）

> 依据：GB 7718-2025 第 4.6、4.9、4.10、4.11 节。与 2011 版差异以 **[新增]** 或 **[变更]** 标注。

## 检查清单

### 一、生产者与经营者基本信息（4.6）

- [ ] **[变更]** 应按照法律法规和国务院食品安全监管部门的规定标示生产者和/或经营者信息（4.6.1）
- [ ] 应标示生产者和/或经营者的名称（4.6.1）
- [ ] 应标示生产者和/或经营者的地址（4.6.1）
- [ ] 应标示生产者和/或经营者的联系方式（4.6.1）
- [ ] 联系方式至少标示一项（电话、传真、网址、邮箱等）（4.6.1）

### 二、委托加工与集团公司

- [ ] 委托生产的食品应标示委托方的名称和地址（4.6.2）
- [ ] 可同时标示受委托方（实际生产者）的名称和地址（4.6.2）
- [ ] 集团公司/子公司/分公司标示应与食品生产许可信息一致（4.6.2）

### 三、进口食品信息（4.6）

- [ ] 进口预包装食品应标示原产国国名或地区区名（4.6.3）
- [ ] 应标示在中国依法登记注册的代理商、进口商或经销者的名称、地址和联系方式（4.6.3）

### 四、食品生产许可证编号（4.9）

- [ ] **[变更]** 食品生产许可证编号按照相关规定执行（4.9）
- [ ] 不再单独规定标示形式，而是援引市场监管部门的相关规定（4.9）

### 五、产品标准代号（4.10）

- [ ] 国内生产并在国内销售的预包装食品应标示产品标准代号（4.10）
- [ ] 产品标准代号包括国家标准、行业标准、地方标准、企业标准、团体标准（4.10）
- [ ] **[新增]** 明确纳入"团体标准"（4.10）

### 六、质量等级（4.11）

- [ ] 产品标准中已明确规定质量等级的食品，应标示质量等级（4.11）
- [ ] **[新增]** 产品标准中未规定质量等级的，不得标示质量等级（4.11）
- [ ] 质量等级应与执行标准中的规定一致（4.11）
