# 日期与保质期审核检查项（GB 7718-2011）

> 依据：GB 7718-2011 第 4.1.7 节，问答（Q&A）第 10、18、57-59 条。

## 检查清单

### 一、基本要求

- [ ] 应清晰标示生产日期和保质期（4.1.7.1）
- [ ] 日期标示不得另外加贴、补印或篡改（4.1.7.1）

### 二、日期格式

- [ ] 应按年、月、日的顺序标示日期（4.1.7.2）
- [ ] 如不按年月日顺序排列，应注明日期标示顺序（4.1.7.2）
- [ ] 年份用四位数字标示，月和日可用一位或两位数字（4.1.7.2）
- [ ] 日期格式示例（4.1.7.2）：
  - [ ] 2011年04月01日
  - [ ] 2011-04-01
  - [ ] 20110401

### 三、日期标示位置

- [ ] 当标示"见包装物某部位"时，应标明具体部位（4.1.7.3）
- [ ] 例如"见瓶盖""见罐底"等，不得仅标"见包装"（Q&A 57）

### 四、保质期标示形式

- [ ] 保质期可按以下形式之一标示（4.1.7.3）：
  - [ ] "保质期至 XXXX年XX月XX日"
  - [ ] "保质期至 XXXXXXXX"
  - [ ] "保质期：XX 个月"（配合生产日期推算）
  - [ ] "保质期：XX 天"
  - [ ] "保质期：XX 年"

### 五、多件包装

- [ ] 外包装上标示的保质期应按最早到期的单件食品保质期计算（4.1.7.4）
- [ ] 也可分别标示各单件食品的生产日期和保质期（4.1.7.4）
- [ ] 外包装上的生产日期应为最早生产的单件食品生产日期；或分别标示（Q&A 59）

### 六、保质期豁免

- [ ] 以下食品可豁免标示保质期（4.1.7.1 + 表2）：
  - [ ] 酒精度 ≥ 10% 的饮料酒
  - [ ] 食醋
  - [ ] 食用盐
  - [ ] 固态食糖类（如白砂糖、冰糖）
  - [ ] 味精

### 七、其他注意事项

- [ ] 保质期与贮存条件应配合标示，保质期受贮存条件影响的应一并标注（Q&A 18）
- [ ] 进口食品生产日期应为该食品的灌装日期或包装日期（Q&A 10）
- [ ] 日期标示应使用与标签其他内容相协调的字体，确保清晰可辨（Q&A 58）
