# 日期与保质期审核检查项（GB 7718-2025）

> 依据：GB 7718-2025 第 4.7 节、10.1-10.2 节。与 2011 版差异以 **[新增]** 或 **[变更]** 标注。

## 检查清单

### 一、基本要求

- [ ] 应清晰标示生产日期和保质期（4.7.1）
- [ ] **[变更]** 应按**年月日顺序**标示生产日期和**保质期到期日**（不再接受仅标"保质期：XX个月"的形式）（4.7.1）
- [ ] 日期标示不得另外加贴、补印或修改（4.7.2）
- [ ] **[新增]** 日期标示不应与包装物/容器分离（4.7.2）

### 二、日期格式

- [ ] 年份用四位数字，月和日分别用两位数字（4.7.3）
- [ ] 日期格式示例（4.7.3）：
  - [ ] 2026年01月01日
  - [ ] 2026-01-01
  - [ ] 20260101
- [ ] 当标示"见包装物某部位"时，应标明具体部位（4.7.4）

### 三、保质期标示简化 **[新增]**

- [ ] **[新增]** 保质期 6 个月及以上的食品，可仅标示保质期和保质期到期日（不强制标生产日期）（4.7.5）
- [ ] **[新增]** 最大表面积 ≤ 20 cm² 的小包装食品，可仅标示保质期和保质期到期日（4.7.5）
- [ ] 上述情形下，若未标生产日期，应标示生产批号（10.1 / 10.2）

### 四、多件包装

- [ ] 外包装上标示的保质期到期日应按最早到期的单件食品计算（4.7.6）
- [ ] 也可分别标示各单件食品的生产日期和保质期到期日（4.7.6）

### 五、消费保存期 **[新增]**

- [ ] **[新增]** 可选标示"消费保存期"：即开封后在推荐贮存条件下的建议食用期限（4.7.7）
- [ ] 消费保存期为可选项，非强制标示内容（4.7.7）

### 六、保质期豁免

- [ ] 以下食品可豁免标示保质期（4.7.1 + 附录 B 表 B.1）：
  - [ ] 酒精度 ≥ 10% 的饮料酒
  - [ ] 食醋
  - [ ] 食用盐
  - [ ] 固态食糖类
  - [ ] 味精
  - [ ] **[新增]** 葡萄酒（单列，不限酒精度）

### 七、生产日期豁免 **[新增]**

- [ ] **[新增]** 葡萄酒可豁免标示生产日期（需已标示批号或年份）（附录 B 表 B.1）
- [ ] **[新增]** 酒精度 ≥ 10% 的其他酒类可豁免标示生产日期（需已标示批号）（附录 B 表 B.1）

### 八、其他注意事项

- [ ] 保质期应与贮存条件配合标示（4.8）
- [ ] 豁免保质期和/或生产日期的食品仍应标示生产批号（10.1）
