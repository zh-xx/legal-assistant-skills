# 营养声称 (GB 28050-2025)

## 检查清单

### 基本规则

- [ ] 营养声称包括**含量声称**和**比较声称**两种
- [ ] 含量声称用语："含有""来源""高""富含""低""无""不含""瘦"等
- [ ] 比较声称用语："增加"或"减少"等
- [ ] 同时符合两种声称要求时，可选其一或同时使用
- [ ] **[新增]** 营养声称、营养成分作用声称的**字高不得超过食品名称的最大字高**
- [ ] 用"份"作为计量单位时，也应符合 100g(mL) 或 420kJ 的含量要求才可声称
- [ ] **[新增]** 半固体或固液混合食品用 100g(mL) 声称时，应与净含量计量单位一致

### 比较声称

- [ ] 与**参考食品**比较，含量变化须达 **25% 以上（含 25%）**
- [ ] **[变更]** 参考食品数据来源：(1) 同一企业同类或同一属类食品；(2)《中国食物成分表》同类食品
- [ ] 适用项目：能量、脂肪、饱和脂肪、糖、钠（减少）；蛋白质、膳食纤维、维生素、矿物质（增加）
- [ ] **[变更]** 2025 版取消了胆固醇、碳水化合物的比较声称

### 声称同义语

| 标准语 | 同义语 |
|--------|--------|
| 无/不含 | 零(0)、没有 |
| 低 | 少 |
| 来源/含有 | 提供、含、有 |
| 高/富含 | 良好来源、含丰富××、丰富(的)××、富 |
| 增加 | 增、加、增高、增加×%(×倍)、多×%、提高×倍 |
| 减少 | 减、减低、降、降低、减少×%、减×%、降×% |

### 与 2011 版核心差异

- 2025 版新增 n-3 多不饱和脂肪酸、乳糖的含量声称
- 2025 版新增"脱脂"声称限定条件（区分液态乳和乳粉）
- 2025 版声称字号规则改为"不得超过食品名称的最大字高"
- 2025 版比较声称取消胆固醇和碳水化合物项目
- 2025 版取消"低蛋白质"声称
- 2025 版含量声称同义语表有简化调整

### 审查要点

- [ ] 声称用语是否使用标准语或其同义语
- [ ] 实际营养素含量是否达到声称阈值（注意使用 2025 版 NRV）
- [ ] 有声称的营养成分是否在营养成分表中标示及 NRV%
- [ ] 声称字高是否不超过食品名称最大字高
