# 净含量与规格审核检查项（GB 7718-2011 & 2025 合并）

> 依据：GB 7718-2011 第 4.1.5 节；GB 7718-2025 第 4.5 节。差异以 **[2025新增]** 或 **[2025变更]** 标注。

## 检查清单

### 一、基本组成

- [ ] 净含量标示由"净含量"（或"净含量："）+ 数字 + 法定计量单位组成（4.1.5.1 / 4.5.1）
- [ ] "净含量"与数字之间可有适当间隔（4.1.5.1 / 4.5.1）
- [ ] **[2025新增]** 以计量方式销售的散装食品除外，不强制标示净含量（4.5.1）

### 二、计量单位选择

- [ ] 液态食品：用体积单位（mL / L）或质量单位（g / kg）标示（4.1.5.2 / 4.5.2）
- [ ] 固态食品：用质量单位（g / kg）标示（4.1.5.2 / 4.5.2）
- [ ] 半固态食品：用质量单位（g / kg）或体积单位（mL / L）标示（4.1.5.2 / 4.5.2）

### 三、计量单位换算阈值

- [ ] 质量：不大于 1000 g 时用 g，大于 1000 g 时用 kg（4.1.5.3 / 4.5.3）
- [ ] 体积：不大于 1000 mL 时用 mL，大于 1000 mL 时用 L（4.1.5.3 / 4.5.3）

### 四、净含量字符最小高度（表 3）

- [ ] 净含量 Q ≤ 50 mL / g：字符高度 ≥ 2 mm（表 3）
- [ ] 50 mL / g < Q ≤ 200 mL / g：字符高度 ≥ 3 mm（表 3）
- [ ] 200 mL / g < Q ≤ 1000 mL / g：字符高度 ≥ 4 mm（表 3）
- [ ] Q > 1000 mL / g：字符高度 ≥ 6 mm（表 3）

### 五、展示位置

- [ ] 净含量应与食品名称在同一展示版面标示（4.1.5.4 / 4.5.4）

### 六、固液两相食品

- [ ] 固、液两相且固相为主要食用成分的食品，除标示净含量外，还应标示沥干物（固形物）含量（4.1.5.5 / 4.5.5）
- [ ] 沥干物含量用质量或质量分数表示（4.1.5.5 / 4.5.5）

### 七、多件包装 / 规格

- [ ] 由多个单件预包装食品组成的包装，应在外包装上标示规格（4.1.5.6 / 4.5.6）
- [ ] 规格可标示为"单件净含量 x 件数"或类似形式（4.1.5.6 / 4.5.6）
- [ ] 也可同时标示总净含量（4.1.5.6 / 4.5.6）
