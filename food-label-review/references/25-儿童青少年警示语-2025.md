# 儿童青少年警示语 (GB 28050-2025 新增)

> 本规则仅适用于 2025 版。2011 版无此要求。

## 检查清单

### 强制要求

- [ ] 预包装食品应在营养成分表**下方**标示："**儿童青少年应避免过量摄入盐油糖**"
- [ ] 该警示语为**强制标示内容**（第 4.5 条）
- [ ] 用语必须完整、准确，不得删改

### 标示位置与格式

- [ ] 标示在营养成分表**下方**
- [ ] 可标示在居左、居中或居右位置
- [ ] 可有框或无框
- [ ] 使用方框表格式时，可在表格底部行内标示
- [ ] 使用文字格式（最大表面 ≤40 cm²）时，在文字末尾标示

### 豁免情形

- [ ] 豁免强制标示营养标签的食品（见 26-营养标签豁免.md）**无需**标示该警示语
- [ ] 但如豁免食品**自愿标示**营养标签，则**应同时标示**该警示语

### 审查要点

- [ ] 是否存在该警示语（非豁免食品为必标项）
- [ ] 用语是否完整准确（"儿童青少年应避免过量摄入盐油糖"）
- [ ] 位置是否在营养成分表下方
