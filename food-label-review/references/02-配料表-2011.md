# 配料表审核检查项（GB 7718-2011）

> 依据：GB 7718-2011 第 4.1.3 节，问答（Q&A）第 21-37、39-45 条。

## 检查清单

### 一、引导词与基本格式

- [ ] 配料表以"配料"或"配料表"为引导词（4.1.3.1）
- [ ] 发酵产品（如酿造类）可使用"原料"或"原料与辅料"作引导词（Q&A 30）
- [ ] 单一配料食品也应标示配料表（Q&A 21）

### 二、配料递减顺序

- [ ] 各配料按制造或加工时的加入量（质量分数）递减顺序排列（4.1.3.1）
- [ ] 加入量不超过 2% 的配料可不按递减顺序排列（4.1.3.1）
- [ ] 复原果蔬汁按还原后的果蔬汁用量排列（Q&A 29）

### 三、复合配料标示

- [ ] 复合配料已有国家标准或行业标准，且加入量 < 25% 时，可直接标示复合配料名称，不展开（4.1.3.1.3）
- [ ] 不符合上述豁免条件时，应在复合配料名称后用括号展开标示其原始配料（4.1.3.1.3）
- [ ] 复合配料中含有的食品添加剂，若在最终产品中起工艺作用，应标示（Q&A 33）

### 四、食品添加剂标示

- [ ] 食品添加剂使用 GB 2760 中的通用名称标示（4.1.3.1.4）
- [ ] 标示形式为以下三种之一（附录 B）：
  - [ ] 形式一：直接标示食品添加剂的具体名称
  - [ ] 形式二：功能类别名称 + INS 号（国际编码）
  - [ ] 形式三：功能类别名称 + 具体名称
- [ ] 附录 B 提供四种整体展示方式（全部标具体名称 / 全部功能+INS / 全部功能+名称 / 混用），选择一种即可
- [ ] 在同一预包装食品标签上，食品添加剂标示形式应一致（4.1.3.1.4）
- [ ] 营养强化剂按 GB 14880 标示（Q&A 36）

### 五、水的标示

- [ ] 加工过程中加入的水应在配料表中标示（4.1.3.1.5）
- [ ] 加工过程中已挥发的水不需要标示（4.1.3.1.5）

### 六、可食用包装物

- [ ] 可食用的包装物（如糯米纸、糖衣）应在配料表中标示（4.1.3.1.6）

### 七、表 1 归类标示

- [ ] 植物油：可标示为"植物油"或"精炼植物油"，如经氢化处理应标"氢化"或"部分氢化"（表 1）
- [ ] 淀粉：可标示为"淀粉"（不包括化学改性淀粉）（表 1）
- [ ] 香辛料：单一品种 ≤ 2% 或合计 ≤ 10% 时可标"香辛料""香辛料类"（表 1）
- [ ] 胶基糖果中胶基糖果：可标"胶基"（表 1）
- [ ] 果脯蜜饯类水果：可标"蜜饯""果脯"（表 1）
- [ ] 食用香精/香料：可标"食用香精""食用香料""食用香精香料"（表 1）

### 八、配料定量标示

- [ ] 食品名称中提及某配料，或标签上强调添加/含有某配料时，应标示该配料的添加量或含量（4.1.4.1）
- [ ] 标签上强调某配料低含量或不含某配料时，应标示该配料在成品中的含量（4.1.4.2）
- [ ] 定量标示以质量分数（%）表示（4.1.4.3）

### 九、特殊情形

- [ ] 甜味剂、防腐剂、着色剂应标示功能类别名称，同时标示具体名称或 INS 号（Q&A 37）
- [ ] 加工助剂不需要标示（4.1.3.1.4 注）
- [ ] 胶原蛋白肠衣属于可食用包装物，应标示（Q&A 42）
- [ ] 酒类产品配料中乙醇（食用酒精）应标示（Q&A 43）
