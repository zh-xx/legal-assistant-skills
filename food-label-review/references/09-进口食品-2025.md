# 09 - 进口食品（GB 7718-2025 第8章）

> **2025大幅扩展，2011版仅有简要规定**

## A. 总体要求

- [ ] 所有可见标示内容（含外文、繁体字）均需符合中国法律法规
- [ ] 必须有印刷或加贴的中文标签

## B. 中外文对应

### B1. 强制性标示内容
- [ ] 强制性标示内容的中文与外文应**一一对应**
- [ ] 不得出现外文有而中文无的强制性信息

### B2. 外文/繁体字
- [ ] 外文所表述的内容应与规范汉字有对应关系
- [ ] 繁体字所表述的内容应与规范汉字有对应关系

### B3. 配料表
- [ ] 外文配料表中的所有配料均应在中文配料表中有对应
- [ ] 中文配料表不得遗漏外文配料表中的任何配料

## C. 生产者与经营者信息

- [ ] 应标示进口商名称
- [ ] 应标示进口商地址
- [ ] 应标示进口商联系方式
- [ ] 或标示代理商名称、地址、联系方式
- [ ] 应标示境外企业注册编号

## D. 原产国/地区

### D1. 标示规则
- [ ] 应标示原产国（地区）名称
- [ ] 按完全获得原则确定原产国
- [ ] 或按实质性改变原则确定原产国

### D2. 灌装/分装情况
- [ ] 灌装国与原产国不一致时，应同时标示灌装国和原产国
- [ ] 分装国与原产国不一致时，应同时标示分装国和原产国

## E. 日期标示

- [ ] 原标签未标示生产日期的，应推算后标示生产日期
- [ ] 保质期标示应符合中国标准要求

## F. 适宜人群/食用量/食用方法

- [ ] 原标签涉及适宜人群信息的，中文标签应有对应标示
- [ ] 原标签涉及食用量信息的，中文标签应有对应标示
- [ ] 原标签涉及食用方法信息的，中文标签应有对应标示

## G. 可不标示的内容

- [ ] 食品生产许可证编号：进口食品可不标示
- [ ] 产品标准代号：进口食品可不标示

## H. 审核检查清单

- [ ] 是否有中文标签（印刷或加贴）
- [ ] 中外文强制性内容是否一一对应
- [ ] 配料表中外文是否完整对应
- [ ] 是否标示进口商/代理商完整信息
- [ ] 是否标示境外企业注册编号
- [ ] 是否正确标示原产国/地区
- [ ] 灌装/分装情况是否同时标示相关国家
- [ ] 生产日期是否已标示或推算标示
- [ ] 原标签涉及人群/用量/方法的是否有中文对应
