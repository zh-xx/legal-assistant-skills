# 营养成分作用声称 (GB 28050-2025)

> **[变更]** 2025 版将"营养成分功能声称"改称"营养成分作用声称"。

## 检查清单

### 基本规则

- [ ] 营养成分作用声称：对某营养成分维持人体正常生长、发育和生理功能作用的描述和说明
- [ ] 前提：该营养成分含量须符合**营养声称**（含量声称或比较声称）的要求和条件
- [ ] **不得对作用声称标准用语进行任何形式的删改、添加和合并**
- [ ] 可使用附录 D 中对应的**一条或多条**标准用语
- [ ] **[新增]** 声称字高**不得超过食品名称的最大字高**

### 与 2011 版主要差异

| 项目 | 2011 版 | 2025 版 |
|------|---------|---------|
| 名称 | 营养成分功能声称 | **营养成分作用声称** |
| 字号限制 | 不大于食品名称和商标 | 不超过食品名称最大字高 |
| 钠-食盐摄入 | 不超过 6g | **不宜超过 5g** |
| 蛋白质 | 5 条用语 | **6 条**（新增"有助于肌肉的增长和维持"） |
| 脂肪 | 5 条用语 | **6 条**（新增"长期高脂肪摄入不利于维持正常体重和体脂水平"） |
| 糖 | 无专属条目 | **新增 2 条**（"糖是提供能量的基本物质""长期高糖摄入可增加龋齿和肥胖发生的风险"） |
| α-亚麻酸 | 无 | **新增**（"α-亚麻酸是人体必需脂肪酸"） |
| 胆固醇 | 有（1 条） | **已删除** |
| 碳水化合物 | 有（4 条） | **已删除** |
| 维生素 A | 2 条 | **4 条** |
| 维生素 B6 | 1 条 | **3 条** |
| 维生素 B12 | 1 条 | **3 条** |
| 维生素 C | 4 条 | **5 条** |
| 维生素 D | 3 条（含"促进钙的吸收"） | **3 条**（改为"促进钙和磷的吸收和利用"） |
| 维生素 K | 无 | **新增**（"是维持正常凝血功能不可缺少的成分"） |
| 钙 | 4 条 | **5 条**（新增"正常的神经和肌肉功能所必需"） |
| 锌 | 3 条 | **4 条**（新增"有助于维持免疫系统的正常生理功能"） |
| 碘 | 1 条 | **2 条**（新增"神经系统发育所必需"） |
| 硒 | 无 | **新增 2 条** |
| 钾 | 无 | **新增 4 条** |
| 磷 | 无 | **新增 2 条** |
| 生物素 | 无 | **新增**（"有助于维持皮肤和毛发健康"） |
| 胆碱 | 无 | **新增**（"是参与脂类代谢的重要成分"） |

### 审查要点

- [ ] 作用声称用语是否与附录 D **原文完全一致**（不得删改、添加、合并）
- [ ] 进行作用声称的营养成分是否同时满足营养声称（含量/比较声称）的阈值要求
- [ ] 是否在营养成分表中标示了该营养成分的含量和 NRV%
- [ ] 声称字高是否不超过食品名称最大字高
