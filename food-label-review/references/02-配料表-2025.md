# 配料表审核检查项（GB 7718-2025）

> 依据：GB 7718-2025 第 4.3 节、4.4 节。与 2011 版差异以 **[新增]** 或 **[变更]** 标注。

## 检查清单

### 一、引导词与基本格式

- [ ] 配料表以"配料"或"配料表"为引导词（4.3.1）
- [ ] 发酵产品可使用"原料"或"原料与辅料"作引导词（4.3.1）
- [ ] 单一配料食品也应标示配料表（4.3.1）

### 二、配料递减顺序

- [ ] 各配料按制造或加工时的加入量**以质量计**递减顺序排列（4.3.2）
- [ ] 加入量不超过 2% 的配料可不按递减顺序排列（4.3.2）

### 三、复合配料标示

- [ ] 复合配料应在其名称后用括号展开标示原始配料（4.3.3）
- [ ] 已有国家标准或行业标准的复合配料，且加入量 < 25%，可不展开（4.3.3）
- [ ] **[新增]** 复合配料中的复合配料可不再继续展开（4.3.3）

### 四、食品添加剂标示

- [ ] 食品添加剂使用 **GB 2760** 中的通用名称标示（4.3.4）
- [ ] 营养强化剂使用 **GB 14880** 中的通用名称标示（4.3.4）
- [ ] **[变更]** 标示形式简化为以下两种之一（4.3.4）：
  - [ ] 形式一：直接标示食品添加剂/营养强化剂的通用名称
  - [ ] 形式二：归类标示——功能类别名称 + 通用名称
- [ ] **[变更]** INS 号（国际编码）仅限**最大表面积 ≤ 60 cm²** 的包装使用（4.3.4 注）
- [ ] **[新增]** 复配食品添加剂应逐一展开标示其中各组分（4.3.4）
- [ ] 在同一预包装食品标签上，食品添加剂标示形式应一致（4.3.4）

### 五、水的标示

- [ ] 加工过程中加入的水应在配料表中标示（4.3.5）
- [ ] 加工过程中已挥发的水不需要标示（4.3.5）

### 六、菌种标示 **[新增]**

- [ ] 使用菌种的食品，应在配料表中标示菌种名称（4.3.6）
- [ ] 菌种名称应按相关规定标示至菌株水平（如适用）（4.3.6）

### 七、可食用包装物

- [ ] 可食用的包装物应在配料表中标示（4.3.7）

### 八、归类标示（表 A.1）

- [ ] 植物油/植物脂肪：可标示为"植物油"或"植物脂肪"，氢化的应标明（表 A.1）
- [ ] 淀粉类：可标示为"淀粉"（不包括化学改性淀粉）（表 A.1）
- [ ] 香辛料：单一品种 ≤ 2% 或合计 ≤ 10% 时可归类标示（表 A.1）
- [ ] 胶基糖果中的胶基物质：可标"胶基"（表 A.1）
- [ ] 果脯蜜饯类水果：可标"蜜饯""果脯"（表 A.1）
- [ ] 食用香精/香料（表 A.1）
- [ ] **[新增]** 酱油：可归类标示为"酱油"（表 A.1）
- [ ] **[新增]** 食醋：可归类标示为"食醋"（表 A.1）
- [ ] **[新增]** 食糖：可归类标示为"食糖"（表 A.1）
- [ ] **[新增]** 淀粉糖（葡萄糖浆、麦芽糖浆等）：可归类标示为"淀粉糖"（表 A.1）
- [ ] **[新增]** 发酵酒/蒸馏酒/食用酒精：可分别归类标示（表 A.1）

### 九、配料定量标示（4.4）

- [ ] **[变更]** 食品名称中提及某种配料时，即需定量标示该配料的添加量或含量（4.4.1）
- [ ] 标签中强调添加或含有某配料时，应定量标示（4.4.1）
- [ ] 标签中强调某配料低含量或不含时，应标示该配料在成品中的含量（4.4.2）
- [ ] **[新增]** 禁止使用"不添加""不使用""未添加""未使用""零添加"等类似用语（4.4.3）
- [ ] **[新增]** "无""不含"等用语要求该配料/物质含量确实为 0（4.4.4）
- [ ] **[新增]** 禁止对法规不允许添加或食品中不应存在的物质使用"无""不含"等用语（4.4.4）
- [ ] **[新增]** 使用香精调配风味的，图片/图案需标注"图案仅供口味参考"或类似说明（4.4.5）
- [ ] 定量标示以质量分数（%）表示（4.4.6）
