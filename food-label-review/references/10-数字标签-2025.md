# 10 - 数字标签（GB 7718-2025 第9章）

> **2025新增章节，2011版无对应内容**

## A. 基本定位

- [ ] 鼓励使用数字标签同时展示标签信息（非强制）
- [ ] 数字标签是实体标签的补充和延伸，不替代实体标签

## B. 内容要求

- [ ] 数字标签内容应符合法律法规及食品安全标准
- [ ] 数字标签内容应与包装上的实体标签信息保持一致
- [ ] 内容不得篡改

## C. 技术手段

- [ ] 应使用符合国家标准的数字化手段（如二维码）
- [ ] 应通过易于获取的形式提供
- [ ] 鼓励兼容多种识别手段（提高可及性）

## D. 标识与提示

- [ ] 应在数字标签入口的临近位置明示
- [ ] 可使用"数字标签"字样
- [ ] 或使用类似含义的字样进行提示

## E. 展示规范

### E1. 一级页面
- [ ] 一级页面应直接展示标签信息
- [ ] 一级页面不应有干扰元素（如广告、弹窗等）

### E2. 可读性
- [ ] 内容应清晰醒目
- [ ] 内容应易于识读

## F. 特殊消费群体

- [ ] 可根据特定消费群体需求提供视频方式
- [ ] 可根据特定消费群体需求提供语音方式
- [ ] 可根据特定消费群体需求提供其他辅助方式

## G. 简化实体标签

- [ ] 使用数字标签标示应标注事项的，可按规定简化包装上的实体标签
- [ ] 简化应符合相关规定的要求

## H. 审核检查清单

- [ ] 若使用数字标签，检查内容是否与实体标签一致
- [ ] 检查是否使用符合国标的数字化手段
- [ ] 检查入口处是否有"数字标签"或类似提示
- [ ] 检查一级页面是否直接展示信息、无干扰元素
- [ ] 检查内容是否清晰醒目、易于识读
- [ ] 若简化了实体标签，检查简化是否符合规定
