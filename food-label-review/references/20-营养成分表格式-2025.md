# 营养成分表格式 — GB 28050-2025

> 依据：GB 28050-2025 第3章 + 附录B
> 与 2011 版差异以 **粗体** 标注

---

## 基本形式

- [ ] 以"方框表"形式标示（特殊情况除外），表头为"营养成分表"
- [ ] 与包装或标签基线垂直

## 含量标示单位

- [ ] 含量以每100g/100mL和/或每份可食部标示
- [ ] 以"份"标示时应在同一版面标明每份质量或体积
- [ ] 份量标示可参考附录E推荐值（18类食品份量参考值）

## 标示位置

- [ ] 标示在最小销售单元包装上

## 格式选择（附录B）

- [ ] 格式1：仅标示强制内容（能量 + 6项核心 + 警示语）
- [ ] 格式2：同时标示每100g(mL)和每份
- [ ] 格式3：标示更多营养成分
- [ ] 格式4：附外文（中文对照外文标示）
- [ ] 格式5：横排格式
- [ ] 格式6：文字格式（适用于小包装）
- [ ] 格式7：附营养声称/功能声称
- [ ] **格式8：多品种格式** ← 2025 新增

## 文字格式适用条件

- [ ] **文字格式仅限最大表面面积 ≤ 40cm²**（2011版为总面积 < 100cm²）

## 儿童青少年警示语（2025 新增）

- [ ] **营养成分表下方必须标示"儿童青少年应避免过量摄入盐油糖"**
- [ ] 可居左、居中或居右排列
- [ ] 可有框线或无框线

## 醒目标示

- [ ] 当标示4.1以外成分时，应使强制标示内容更加醒目
- [ ] 醒目方式：增大字号/斜体/加粗/变色等

---

## 与 2011 版核心差异

| 项目 | 2011 | 2025 |
|------|------|------|
| 文字格式条件 | 总面积 < 100cm² | **最大表面面积 ≤ 40cm²** |
| 儿童警示语 | 无 | **强制标示** |
| 格式种类 | 6种 | **8种（新增每份+多品种）** |
| 份量参考 | 无 | **附录E 18类参考值** |
| 醒目要求 | 能量+核心需醒目 | 标更多成分时强制标示内容需醒目 |
