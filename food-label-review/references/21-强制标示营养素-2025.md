# 强制标示营养素 — GB 28050-2025

> 依据：GB 28050-2025 第4章
> 与 2011 版差异以 **粗体** 标注

---

## 强制标示项（"6+能量+警示语"）

- [ ] 能量 (kJ)
- [ ] 蛋白质 (g)
- [ ] 脂肪 (g)
- [ ] **饱和脂肪（或饱和脂肪酸）(g)** ← 2025 新增必标
- [ ] 碳水化合物 (g)
- [ ] **糖 (g)** ← 2025 新增必标
- [ ] 钠 (mg)
- [ ] 以上各项均需标示 NRV%
- [ ] **"儿童青少年应避免过量摄入盐油糖"** ← 2025 新增强制警示语，标示在营养成分表下方

## 额外强制标示条件

- [ ] 对4.1以外营养成分进行声称时需标示
- [ ] 使用营养强化剂时需标示
- [ ] 配料/生产过程使用氢化/部分氢化油脂时需标示反式脂肪酸

## 表达方式

- [ ] 名称、顺序、单位、修约间隔、"0"界限值应符合相关规定
- [ ] 含量只能使用具体数值，不能用范围值
- [ ] 含量 ≤"0"界限值时应标示为"0"

---

## 与 2011 版核心差异总结

| 项目 | 2011 | 2025 |
|------|------|------|
| 核心营养素 | 4项（蛋白质/脂肪/碳水/钠） | **6项（+饱和脂肪+糖）** |
| 儿童警示语 | 无 | **强制** |
| 反式脂肪酸 | 使用氢化油时强制 | 同 |
| 强制标示总结 | "4+1"（4项核心+能量） | **"6+能量+警示语"** |

## 审查要点速查

| 检查项 | 合规标准 |
|--------|----------|
| 必标项数量 | 能量 + 6项核心营养素 + 警示语 |
| 新增：饱和脂肪 | 必须标示，单位 g，需标 NRV% |
| 新增：糖 | 必须标示，单位 g，需标 NRV% |
| 新增：儿童警示语 | "儿童青少年应避免过量摄入盐油糖" |
| 警示语位置 | 营养成分表下方 |
| 声称触发 | 声称4.1以外成分 → 必须标示 |
| 营养强化剂 | 使用 → 必须标示 |
| 氢化油脂 | 使用 → 必须标示反式脂肪酸 |
