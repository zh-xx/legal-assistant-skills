# 营养标签豁免

## 2011 版豁免条件（满足任一即可）

- [ ] 生鲜食品（包装的生肉、生鱼、生蔬菜水果、禽蛋等）
- [ ] 乙醇含量 ≥ 0.5% 的饮料酒类
- [ ] 包装总表面积 ≤ 100 cm² 或最大表面面积 ≤ 20 cm²
- [ ] 现制现售食品
- [ ] 包装饮用水
- [ ] 每日食用量 ≤ 10g 或 10mL 的预包装食品
- [ ] 其他法律法规标准规定可不标示的

## 2025 版豁免条件（满足任一即可）

- [ ] 生鲜食品和**粮食籽粒**（畜禽肉、鱼虾蟹贝、鲜蛋、蔬菜水果、菌藻类等）
- [ ] **经简单物理处理、未添加其他配料的单一原料干制品**（如谷物和杂粮等）
- [ ] 包装饮用水、**茶叶**
- [ ] 酒精度 ≥ 0.5%vol 的饮料酒
- [ ] 每日食用量 ≤ 10g(mL) 的预包装食品和**单一原料调味品**
- [ ] 包装物或包装容器的**最大表面面积 ≤ 40 cm²**
- [ ] 其他法律法规和食品安全国家标准规定可不标示的

## 2011 vs 2025 主要差异

| 项目 | 2011 版 | 2025 版 |
|------|---------|---------|
| 生鲜食品 | 生鲜食品 | 生鲜食品 + 粮食籽粒 |
| 干制品 | 未涉及 | **新增**：单一原料干制品 |
| 茶叶 | 未涉及 | **新增** |
| 面积阈值 | 总面积 ≤100cm² 或最大面 ≤20cm² | 最大表面面积 ≤40cm² |
| 调味品 | 未涉及 | **新增**：单一原料调味品 |
| 现制现售 | 明确列出 | 2025 版未再单列（不适用于 GB 7718） |

## 重要提醒

- [ ] 豁免强制标示营养标签的食品，如在包装上**出现任何营养信息**，仍应按标准执行
- [ ] 特殊膳食用食品营养标签按 **GB 13432** 执行，不在本标准豁免范围内

## 审查要点

- [ ] 确认产品是否符合豁免条件（版本对应）
- [ ] 如已豁免但自愿标示营养信息，是否仍符合标准要求
- [ ] 如不符合任何豁免条件，营养标签为强制标示
