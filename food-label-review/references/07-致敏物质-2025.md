# 07 - 致敏物质（GB 7718-2025 第4.12条 + 附录D）

> **性质：强制标示（从2011版"推荐"升级为"强制"！这是重大变化）**

## A. 八类致敏物质清单（与2011相同）

- [ ] a) 含麸质的谷物及其制品（小麦、黑麦、大麦、燕麦、斯佩耳特小麦或它们的杂交品系）
- [ ] b) 甲壳纲类动物及其制品（虾、龙虾、蟹等）
- [ ] c) 鱼类及其制品
- [ ] d) 蛋类及其制品
- [ ] e) 花生及其制品
- [ ] f) 大豆及其制品
- [ ] g) 乳及乳制品（包括乳糖）
- [ ] h) 坚果及其果仁类制品

## B. 强制标示方式（二选一）

### 方式一：配料表中标示
- [ ] 在配料表中使用易辨识的名称标示致敏物质
- [ ] 可使用字体**加粗**标示以增强辨识度
- [ ] 可使用<u>下划线</u>标示以增强辨识度

### 方式二：配料表临近位置专门标示
- [ ] 在配料表临近位置设置专门的提示信息
- [ ] 引导词可使用：
  - "食物致敏原"
  - "食物过敏原"
  - "致敏物质"
  - "过敏物质"
  - 或其他类似表述

## C. 间接带入（共用车间/生产线）

- [ ] 因共用生产车间或生产线可能带入致敏物质的：**鼓励**标示（非强制）
- [ ] 标示方式参照方式二，在临近位置提示

## D. 免于标示的情况（附录D.4）

### D4. 以下致敏物质来源的配料/加工助剂免于标示：
- [ ] 精炼大豆油
- [ ] 精炼花生油
- [ ] 大豆来源的肽
- [ ] 大豆来源的磷脂
- [ ] 大豆来源的维生素E（生育酚）
- [ ] 淀粉 / 糊精 / 葡萄糖浆（谷物来源）
- [ ] 谷物来源的精炼植物油
- [ ] 甲壳素
- [ ] 壳寡糖
- [ ] 鱼明胶
- [ ] 精炼鱼油
- [ ] 乳糖醇

### D5. 其他免于标示的情况：
- [ ] 单一配料即为致敏物质，且产品名称已明确标示该致敏物质名称的
- [ ] 食用酒精
- [ ] 蒸馏酒

## E. 审核要点（与2011关键差异）

| 对比项 | 2011 | 2025 |
|--------|------|------|
| 标示性质 | 推荐 | **强制** |
| 标示增强手段 | 未规定 | 可用加粗/下划线 |
| 引导词规定 | 未明确 | 明确可用"致/过敏原""致/过敏物质"等 |
| 间接带入 | 宜标示 | 鼓励标示 |
| 免标清单 | 未规定 | 附录D.4/D.5明确列出 |
