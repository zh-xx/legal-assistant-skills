# 08 - 食品声称（GB 7718-2025 第7章）

> **2025新增章节，2011版无对应内容**

## A. 基本原则

- [ ] 食品声称应真实、准确描述食品/配料/成分的特征、特性与特点
- [ ] 法律法规、食品安全标准、国务院相关部门有明确规定的，应从其规定

## B. 与配料定量标示(4.4)的联动

### B1. "无""不含"声称
- [ ] 使用"无""不含"声称的，该物质含量应为0
- [ ] 例外：其他法律法规或食品安全标准另有规定的除外
- [ ] 审核要点：核查声称物质的实际含量是否确为0

### B2. 禁止对不允许添加/不应存在的物质使用"无""不含"
- [ ] 食品中不允许添加的物质，不得使用"无""不含"声称
- [ ] 食品中不应存在的物质，不得使用"无""不含"声称
- [ ] 示例：不得标示"无三聚氰胺"（该物质本就不允许添加）

### B3. 禁止使用"不添加""不使用"及其同义语
- [ ] **禁止**使用"不添加"声称
- [ ] **禁止**使用"不使用"声称
- [ ] **禁止**使用上述词语的同义语
- [ ] 例外：其他法律法规或食品安全标准另有规定的除外
- [ ] 同义语清单详见附录 A.5

## C. 附录 A.5 同义语参考

- [ ] "不添加"的同义语包括但不限于：
  - "零添加"
  - "未添加"
  - "不加入"
  - "不使用"
  - "未使用"
  - 以及其他表达相同含义的词语

## D. 审核检查清单

- [ ] 检查标签上是否存在任何食品声称
- [ ] 若有声称，核查是否真实准确
- [ ] 若有"无""不含"声称，核查含量是否为0
- [ ] 若声称涉及本就不允许添加/不应存在的物质，判定违规
- [ ] 若使用"不添加""不使用"或其同义语，判定违规（除非有其他法规允许）
- [ ] 若声称涉及法规/标准已有规定的内容，核查是否符合相应规定
