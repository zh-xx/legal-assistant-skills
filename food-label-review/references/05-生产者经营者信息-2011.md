# 生产者与经营者信息审核检查项（GB 7718-2011）

> 依据：GB 7718-2011 第 4.1.6、4.1.9、4.1.10 节，问答（Q&A）第 50-54、60 条。

## 检查清单

### 一、生产者基本信息（4.1.6）

- [ ] 应标注生产者的名称（4.1.6.1）
- [ ] 应标注生产者的地址（4.1.6.1）
- [ ] 应标注生产者的联系方式（4.1.6.1）
- [ ] 联系方式至少标示以下一项：电话、传真、网址、邮箱、邮政地址（Q&A 50）

### 二、集团公司与关联企业标示（Q&A 51-52）

- [ ] 集团公司与子公司：可同时标示集团公司和子公司名称、地址，实际生产者应明确标示（Q&A 51）
- [ ] 集团公司与分公司：分公司不具法人资格的，应标示集团公司名称和分公司地址（Q&A 51）
- [ ] 标示的生产者信息应当真实、与生产许可证内容一致（Q&A 52）

### 三、委托加工标示（4.1.6.1 + Q&A 53-54）

- [ ] 依法承担法律责任的生产者或经营者名称和地址应标示（4.1.6.1）
- [ ] 委托加工：应标示委托方的名称和地址，可同时标示受委托方名称和地址（Q&A 53）
- [ ] 委托加工标示应符合相关资质与许可要求（Q&A 54）
- [ ] 受委托方具有独立食品生产许可证的，可单独标示受委托方信息（Q&A 54）

### 四、进口食品信息（4.1.6.2）

- [ ] 进口预包装食品应标示原产国国名或地区区名（4.1.6.2）
- [ ] 应标示在中国依法登记注册的代理商、进口商或经销者的名称、地址和联系方式（4.1.6.2）

### 五、食品生产许可证编号（4.1.9）

- [ ] 国内生产并在国内销售的预包装食品应标示食品生产许可证编号（4.1.9）
- [ ] 编号标示位置应醒目（Q&A 60）

### 六、产品标准代号（4.1.10.1）

- [ ] 国内生产并在国内销售的预包装食品应标示产品标准代号（4.1.10.1）
- [ ] 产品标准代号包括国家标准、行业标准、地方标准、企业标准等（4.1.10.1）

### 七、质量等级（4.1.10.2）

- [ ] 产品标准中已明确规定质量等级的食品，应标示质量等级（4.1.10.2）
- [ ] 质量等级应与执行标准中的规定一致（4.1.10.2）
