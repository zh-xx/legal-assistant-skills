# 营养声称 (GB 28050-2011)

## 检查清单

### 基本规则

- [ ] 营养声称包括**含量声称**和**比较声称**两种
- [ ] 含量声称：描述能量或营养成分含量水平（如"低""无""含有""高"）
- [ ] 比较声称：与同类食品比较含量变化（如"增加""减少"）
- [ ] 同时符合含量声称和比较声称要求时，可选其一或同时使用
- [ ] 营养声称、营养成分功能声称的**字号不得大于食品名称和商标**
- [ ] 用"份"作为计量单位时，也应符合 100g(mL) 的含量要求才可声称

### 含量声称阈值速查

| 项目 | 声称方式 | 含量要求 | 限制条件 |
|------|---------|---------|---------|
| 能量 | 无能量 | ≤17 kJ/100g 或 100mL | 脂肪供能 ≤50% |
| | 低能量 | ≤170 kJ/100g；≤80 kJ/100mL | 同上 |
| 蛋白质 | 来源/含有 | ≥10% NRV/100g；≥5% NRV/100mL；≥5% NRV/420kJ | |
| | 高/富含 | ≥20% NRV/100g；≥10% NRV/100mL；≥10% NRV/420kJ | |
| 脂肪 | 无/不含 | ≤0.5 g/100g 或 100mL | |
| | 低脂肪 | ≤3 g/100g；≤1.5 g/100mL | |
| 饱和脂肪 | 无/不含 | ≤0.1 g/100g 或 100mL | 指饱和脂肪+反式脂肪总和 |
| | 低饱和脂肪 | ≤1.5 g/100g；≤0.75 g/100mL | 供能比 <10% |
| 反式脂肪酸 | 无/不含 | ≤0.3 g/100g 或 100mL | |
| 糖 | 无/不含 | ≤0.5 g/100g 或 100mL | |

| 钠 | 无/不含 | ≤5 mg/100g 或 100mL | 可用"盐"代替"钠" |

### 比较声称

- [ ] 与**参考食品**比较，含量变化须达 **25% 以上**
- [ ] 参考食品须为消费者熟知、容易理解的**同类或同一属类食品**
- [ ] 适用项目：能量、蛋白质、脂肪、胆固醇、碳水化合物、糖、膳食纤维、钠、矿物质、维生素

### 声称同义语

| 标准语 | 同义语 |
|--------|--------|
| 不含/无 | 零(0)、没有、100%不含 |
| 低 | 少、少油(仅用于低脂肪) |
| 含有/来源 | 提供、含、有 |
| 富含/高 | 良好来源、含丰富××、丰富(的)×× |
| 增加 | 增、加、增高、添加(了)、多×%、提高×倍 |
| 减少 | 减、少、减低、降、降低 |

### 审查要点

- [ ] 声称用语是否使用标准语或其同义语
- [ ] 实际营养素含量是否达到声称阈值
- [ ] 有声称的营养成分是否在营养成分表中标示
- [ ] 声称字号是否不大于食品名称和商标
