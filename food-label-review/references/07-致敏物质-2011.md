# 07 - 致敏物质（GB 7718-2011 第4.4.3条 + 问答62）

> **性质：推荐标示（非强制）**

## A. 八类致敏物质清单

- [ ] a) 含麸质的谷物及其制品（小麦、黑麦、大麦、燕麦、斯佩耳特小麦或它们的杂交品系）
- [ ] b) 甲壳纲类动物及其制品（虾、龙虾、蟹等）
- [ ] c) 鱼类及其制品
- [ ] d) 蛋类及其制品
- [ ] e) 花生及其制品
- [ ] f) 大豆及其制品
- [ ] g) 乳及乳制品（包括乳糖）
- [ ] h) 坚果及其果仁类制品

## B. 标示方式

### B1. 配料表中直接标示
- [ ] 在配料表中使用易识别的名称直接标示致敏物质
- [ ] 示例：配料中直接写明"小麦粉""牛奶""花生"等易识别名称

### B2. 配料表邻近位置提示
- [ ] 或在配料表邻近位置加以提示
- [ ] 示例："含有小麦、牛奶、花生"

## C. 非直接添加但可能带入的致敏物质

- [ ] 对于生产过程中可能带入的致敏物质（交叉污染风险）
- [ ] 宜在配料表邻近位置加以提示
- [ ] 示例："本生产设备还加工含有花生的产品"
- [ ] 示例："可能含有微量虾成分"

## D. 审核要点

- [ ] 确认标示为"推荐"性质，未标示致敏信息不构成违规
- [ ] 若已标示致敏信息，检查所列致敏物质是否与配料一致
- [ ] 若已标示致敏信息，检查名称是否使用易识别的通用名称
- [ ] 若已标示致敏信息，检查标示位置是否在配料表中或其邻近位置
