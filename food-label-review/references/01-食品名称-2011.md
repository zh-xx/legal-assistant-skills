# 食品名称审核 (GB 7718-2011)

## 检查清单

### 属性名称
- [ ] 在食品标签的**醒目位置**清晰标示反映食品真实属性的专用名称
- [ ] 有国标/行标/地标规定名称时，选用其中一个或等效名称
- [ ] 无标准规定名称时，使用不使消费者误解或混淆的常用名称或通俗名称

### 商标名称/新创名称
- [ ] 标示"新创名称""奇特名称""音译名称""牌号名称""地区俚语名称""商标名称"时，应在**同一展示版面**同时标示反映真实属性的专用名称
- [ ] 当上述名称含有**易使人误解食品属性**的文字或术语时，应在同一展示版面**邻近部位**使用**同一字号**标示食品真实属性的专用名称
- [ ] 当食品真实属性的专用名称因**字号或字体颜色不同**易使人误解时，也应使用同一字号及同一字体颜色标示

### 状态/工艺描述
- [ ] 为避免消费者误解食品的真实属性、物理状态或制作方法，可在食品名称前后附加说明词（如干燥的、浓缩的、复原的、熏制的、油炸的、粉末的、粒状的等）

### 常见问题 (Q&A)

**Q19：什么是反映食品真实属性的专用名称？**
通常是国标/行标/地标中规定的食品名称或食品分类名称。若有多个可选其一。能反映食品本身固有的性质、特性、特征，具有区分不同产品的作用。

**Q20：商品名称易使人误解时如何处理？**
应在同一展示版面邻近部位使用同一字号标示食品真实属性的专用名称。如因字号或字体颜色不同而易使人误解，应使用同一字号及同一字体颜色标示。

**Q11：仅添加了香精香料而非该种食品配料，能否标示该食品实物图案？**
不应使用易使消费者误解或欺骗性的图形。若使用的图形或文字可能使消费者误解时，应用清晰醒目的文字加以说明。
