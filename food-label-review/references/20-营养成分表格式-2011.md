# 营养成分表格式 — GB 28050-2011

> 依据：GB 28050-2011 第3章基本要求 + 附录B + 问答17-19, 73-78

---

## 基本形式

- [ ] 以"方框表"形式标示（特殊情况除外），表题为"营养成分表"
- [ ] 方框可为任意尺寸，与包装基线垂直
- [ ] 基本要素：表头、营养成分名称、含量、NRV%、方框

## 含量标示单位

- [ ] 含量以每100g/100mL和/或每份可食部标示
- [ ] 以"份"标示时应标明每份食品的量
- [ ] 含量数值应为可食部的含量

## 标示位置

- [ ] 标示在最小销售单元包装上

## 格式选择（附录B 6种格式）

- [ ] 格式1：仅标示核心营养素（能量 + 4项核心）
- [ ] 格式2：标示更多营养成分（核心 + 可选项）
- [ ] 格式3：附外文（中文对照外文标示）
- [ ] 格式4：横排格式
- [ ] 格式5：文字格式（适用于小包装）
- [ ] 格式6：附营养声称/功能声称

## 文字格式适用条件

- [ ] 文字格式仅限包装总表面积 < 100cm² 的食品使用
- [ ] 使用文字格式时可省略 NRV%

## 文字与语言

- [ ] 使用中文标示，可同时标示外文
- [ ] 外文字号不得大于中文字号

## 醒目标示

- [ ] 能量和核心营养素应采取适当形式使其醒目（加粗/加大/变色等）

---

## 审查要点速查

| 检查项 | 合规标准 |
|--------|----------|
| 表题 | 必须为"营养成分表" |
| 表形式 | 方框表（小包装可用文字格式） |
| 方向 | 与包装基线垂直 |
| 含量单位 | 每100g/100mL 和/或 每份 |
| 每份标示 | 需标明每份食品的量 |
| 小包装文字格式 | 总面积 < 100cm² |
| 外文字号 | 不大于中文 |
| 醒目要求 | 能量+核心营养素需醒目 |
