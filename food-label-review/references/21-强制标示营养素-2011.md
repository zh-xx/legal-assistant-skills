# 强制标示营养素 — GB 28050-2011

> 依据：GB 28050-2011 第4章 + 问答20, 29-30

---

## 强制标示项（"4+1"）

- [ ] 能量 (kJ)
- [ ] 蛋白质 (g)
- [ ] 脂肪 (g)
- [ ] 碳水化合物 (g)
- [ ] 钠 (mg)
- [ ] 以上各项均需标示 NRV%

## 额外强制标示条件

- [ ] 对能量和核心营养素外的其他营养成分进行声称时，需在营养成分表中标示该成分
- [ ] 使用了营养强化剂时，需标示强化后含量及 NRV%
- [ ] 配料含有或生产过程使用了氢化/部分氢化油脂时，需标示反式脂肪（酸）含量

## 表达方式

- [ ] 名称、顺序、单位、修约间隔、"0"界限值应符合表1规定
- [ ] 含量只能使用具体数值，不能用范围值（如 ≤XX、≥XX）
- [ ] 含量 ≤"0"界限值时应标示为"0"

---

## 审查要点速查

| 检查项 | 合规标准 |
|--------|----------|
| 必标项数量 | 能量 + 4项核心营养素 = "4+1" |
| 能量单位 | kJ（千焦） |
| 蛋白质单位 | g |
| 脂肪单位 | g |
| 碳水化合物单位 | g |
| 钠单位 | mg |
| NRV% | 所有必标项均需标示 |
| 声称触发 | 声称某成分 → 必须标示该成分 |
| 营养强化剂 | 使用 → 必须标示强化后含量+NRV% |
| 氢化油脂 | 使用 → 必须标示反式脂肪（酸） |
| 含量表达 | 具体数值，不可用范围值 |
| "0"标示 | 含量 ≤ "0"界限值时标示为"0" |
| 名称/顺序/单位 | 严格按照表1规定 |
