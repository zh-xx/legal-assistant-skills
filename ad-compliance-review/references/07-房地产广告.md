# 房地产广告

## 房源基本信息

- [ ] 面积应当明确表明为**建筑面积**或**套内建筑面积**
- [ ] 房源信息真实存在

## 禁止内容（红线）

- [ ] 未承诺**升值或投资回报**
- [ ] 未以到达某参照物的**所需时间**表示项目位置（如"距XX 10分钟车程"）
- [ ] 未对**规划或建设中**的交通、商业、教育设施等作误导宣传
- [ ] 未违反国家价格管理规定
- [ ] 未含有风水、占卜等**封建迷信内容**
- [ ] 未利用**其他项目的形象、环境**作为本项目效果
- [ ] 未出现**融资或变相融资**内容
- [ ] 未承诺可办理**户口、就业、升学**事项

## 资质与证明文件

- [ ] 已取得国有土地使用权
- [ ] 不在司法机关/行政机关查封或限制权利状态
- [ ] **预售项目**已取得预售许可证
- [ ] 权属无争议
- [ ] 已通过竣工验收合格
- [ ] 具有房地产开发企业资质证书

## 广告必载事项

- [ ] 载明**开发企业名称**
- [ ] 中介代理的，载明**中介机构名称**
- [ ] 载明**预售或销售许可证书号**

## 特殊标注要求

- [ ] 涉及价格的，清楚表示为**实际销售价格**，并明示**价格有效期限**
- [ ] 使用**建筑设计效果图或模型照片**的，已在广告中注明
- [ ] 涉及规划/建设中的设施的，已**注明"规划中"或"建设中"**
- [ ] 涉及贷款服务的，载明**银行名称、贷款额度、年期**
- [ ] 涉及价格评估的，表明**评估单位、估价师和评估时间**
- [ ] 项目位置示意图**准确、清楚、比例恰当**

## 禁止销售方式

- [ ] 未采取**返本销售或变相返本销售**
- [ ] 未采取**售后包租或变相售后包租**（未竣工商品房）
- [ ] 未对商品住宅**分割拆零销售**
