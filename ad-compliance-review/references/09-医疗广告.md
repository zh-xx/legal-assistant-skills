# 医疗广告

## 审查与资质

- [ ] 未经审查不得发布医疗广告
- [ ] 已取得《医疗广告审查证明》并在广告中显著标明文号
- [ ] 非医疗机构未发布医疗广告
- [ ] 未以内部科室名义发布医疗广告

## 内容限制（需与《医疗机构执业许可证》一致）

- [ ] 医疗机构第一名称
- [ ] 医疗机构地址
- [ ] 所有制形式
- [ ] 医疗机构类别
- [ ] 诊疗科目
- [ ] 床位数
- [ ] 接诊时间
- [ ] 联系电话

## 禁止内容

- [ ] 未涉及医疗技术、诊疗方法、疾病名称、药物
- [ ] 未保证治愈或隐含保证治愈
- [ ] 未宣传治愈率、有效率等诊疗效果
- [ ] 未利用患者、卫生技术人员、医学教育科研机构及人员等名义、形象作证明
- [ ] 未使用解放军和武警部队名义
- [ ] 未利用新闻形式、医疗资讯服务类专题节（栏）目发布或变相发布医疗广告
