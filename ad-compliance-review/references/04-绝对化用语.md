# 绝对化用语

## 禁止用语

- [ ] 未使用"国家级、最高级、最佳、唯一、最安全、最有效"等
- [ ] 未使用近似或变体表达

## 例外条件（如使用绝对化表达，是否满足以下之一）

- [ ] 仅表达理念、愿望、目标
- [ ] 企业或品牌内部对比
- [ ] 有明确限定条件的客观事实
- [ ] 有国家 / 行业 / 地方标准依据
- [ ] 属于注册商标、专利名称本身

## 重点领域（从严审核）

以下领域须完全避免与效果、收益、治愈率相关的绝对化表述：

- [ ] 医疗 / 医美 / 药品 / 医疗器械 / 保健食品
- [ ] 投资 / 招商 / 理财
- [ ] 教育 / 培训
