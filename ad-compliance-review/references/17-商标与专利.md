# 商标与专利规范

## 商标使用规范

- [ ] 未在商品、商品包装或容器上使用"驰名商标"字样
- [ ] 未在广告宣传、展览以及其他商业活动中使用"驰名商标"字样
- [ ] 经许可使用他人注册商标的，在使用该注册商标的商品上标明被许可人的名称和商品产地
- [ ] 注册商标标明"注册商标"或注册标记
- [ ] 未将未注册商标冒充注册商标使用
- [ ] 未使用与他人注册商标相同或近似的商标
- [ ] 未使用禁用标志（国家名称、国旗、国徽等）
- [ ] 未使用带有欺骗性、容易使公众误认的标志
- [ ] 未将与他人注册商标相同或近似的标志作为企业名称中的字号使用，误导公众

## 专利广告规范

- [ ] 广告中涉及专利产品或专利方法的，标明专利号和专利种类
- [ ] 未取得专利权的，未在广告中谎称取得专利权
- [ ] 未在广告中称未授予专利权的专利申请为专利产品或专利方法
- [ ] 未使用已经终止、撤销、无效的专利作广告
- [ ] 未将非专利产品冒充专利产品或将非专利方法冒充专利方法
