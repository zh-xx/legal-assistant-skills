# 促销 / 有奖销售

## 赠送 / 促销

- [ ] 明确赠送品种、规格、数量、期限、方式
- [ ] 赠品合法、不侵权
- [ ] 折扣基于合法基准价
- [ ] 承诺可兑现
- [ ] 开展价格促销活动有附加条件的，显著标明条件
- [ ] 开展限时减价、折价等价格促销活动的，显著标明期限
- [ ] 折价、减价标明或表明折价、减价的基准

## 有奖销售信息公示

- [ ] 明确公布奖项种类、参与条件、参与方式、开奖时间、开奖方式
- [ ] 公布奖金金额或奖品价格、奖品品名、奖品种类、奖品数量或中奖概率
- [ ] 公布兑奖时间、兑奖条件、兑奖方式、奖品交付方式、弃奖条件
- [ ] 公布主办方及其联系方式
- [ ] 现场即时开奖活动中，超过500元奖项的兑奖情况随时公示

## 奖品形式规范

- [ ] 奖品为积分、礼券、兑换券、代金券等形式的，公布兑换规则、使用范围、有效期限及其他限制性条件
- [ ] 需向其他经营者兑换的，公布其他经营者名称、兑换地点或兑换途径

## 禁止行为

- [ ] 未采用虚构奖项、奖品、奖金金额等谎称有奖方式
- [ ] 未仅在特定区域投放奖品
- [ ] 未将带有中奖标志的商品、奖券不投放或未全部投放市场
- [ ] 未将不同奖金金额的商品、奖券按不同时间投放市场
- [ ] 未按明示信息兑奖
- [ ] 未让内部员工、指定单位或个人中奖等故意让内定人员中奖

## 奖品金额限制

- [ ] 抽奖式有奖销售最高奖金额未超过5万元
- [ ] 非现金形式奖品按同期市场同类商品价格计算金额
- [ ] 以物品使用权、服务等形式作为奖品的，市场价格未超过5万元
- [ ] 以游戏装备、账户等网络虚拟物品作为奖品的，市场价格未超过5万元
- [ ] 以降价、优惠、打折等方式作为奖品的，折算价格未超过5万元
- [ ] 以彩票、抽奖券等作为奖品的，可能最高奖金额未超过5万元

## 档案管理

- [ ] 建立档案，如实记录设奖规则、公示信息、兑奖结果、获奖人员等内容
- [ ] 档案留存期限不少于 2 年
