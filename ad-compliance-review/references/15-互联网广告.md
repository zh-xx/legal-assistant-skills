# 互联网广告

## 广告识别与标识

- [ ] 竞价排名的商品或服务显著标明"广告"，与自然搜索结果明显区分
- [ ] 通过知识介绍、体验分享、消费测评等形式推销商品或服务并附加购物链接的，显著标明"广告"
- [ ] 介绍健康、养生知识的页面，未同时出现相关医疗、药品、医疗器械、保健食品、特殊医学用途配方食品的商品经营者或服务提供者地址、联系方式、购物链接等内容

## 弹出广告规范

- [ ] 显著标明关闭标志，确保一键关闭
- [ ] 不存在没有关闭标志或计时结束才能关闭广告的情形
- [ ] 不存在关闭标志虚假、不可清晰辨识或难以定位等障碍
- [ ] 关闭广告不需要两次以上点击
- [ ] 在浏览同一页面、同一文档过程中，关闭后不继续弹出广告
- [ ] 启动互联网应用程序时的开屏广告符合一键关闭要求

## 禁止欺骗误导用户点击

- [ ] 不存在虚假的系统或软件更新、报错、清理、通知等提示
- [ ] 不存在虚假的播放、开始、暂停、停止、返回等标志
- [ ] 不存在虚假的奖励承诺
- [ ] 无其他欺骗、误导用户点击、浏览广告的方式

## 未成年人保护

- [ ] 在针对未成年人的网站、网页、互联网应用程序、公众号等互联网媒介上，未发布医疗、药品、保健食品、特殊医学用途配方食品、医疗器械、化妆品、酒类、美容广告
- [ ] 在针对未成年人的互联网媒介上，未发布不利于未成年人身心健康的网络游戏广告

## 用户同意与个人信息保护

- [ ] 未经用户同意、请求或用户明确表示拒绝的，未向其交通工具、导航设备、智能家电等发送互联网广告
- [ ] 未在用户发送的电子邮件或互联网即时通讯信息中附加广告或广告链接
- [ ] 未影响用户正常使用网络
- [ ] 未在搜索政务服务网站、网页、互联网应用程序、公众号等的结果中插入竞价排名广告

## 广告档案管理

- [ ] 广告主自行发布互联网广告的，建立广告档案并及时更新（留存期限自广告发布行为终了之日起不少于3年）
- [ ] 广告经营者、广告发布者建立、健全和实施互联网广告业务的承接登记、审核、档案管理制度
- [ ] 查验并登记广告主的真实身份、地址和有效联系方式等信息
- [ ] 配备熟悉广告法律法规的广告审核人员或设立广告审核机构
- [ ] 利用算法推荐等方式发布互联网广告的，将其算法推荐服务相关规则、广告投放记录等记入广告档案

## 互联网信息服务提供者义务

- [ ] 记录、保存利用其信息服务发布广告的用户真实身份信息（留存期限自广告发布行为终了之日起不少于3年）
- [ ] 对利用其信息服务发布的广告内容进行监测、排查
- [ ] 建立有效的投诉、举报受理和处置机制
- [ ] 依据服务协议和平台规则对发布违法广告的用户采取警示、暂停或终止服务等措施

## 网络直播营销特别规定

- [ ] 直播间运营者账号名称、头像、简介，直播间标题、封面，直播间布景、道具、商品展示，直播营销人员着装、形象符合法律法规和国家有关规定
- [ ] 直播营销人员年满十六周岁（十六周岁以上的未成年人经监护人同意）
- [ ] 未营销假冒伪劣、侵犯知识产权或不符合保障人身、财产安全要求的商品
- [ ] 未虚构或篡改交易、关注度、浏览量、点赞量等数据流量造假
- [ ] 未在涉及国家安全、公共安全、影响他人及社会正常生产生活秩序的场所从事网络直播营销活动
- [ ] 使用其他人肖像作为虚拟形象从事网络直播营销活动的，征得肖像权人同意
- [ ] 直播营销平台对利用人工智能、数字视觉、虚拟现实、语音合成等技术展示的虚拟形象进行安全评估并以显著方式予以标识
