# 农药广告

## 审查与资质

- [ ] 未经国家批准登记的农药未发布广告
- [ ] 内容与《农药登记证》和《农药登记公告》内容相符

## 禁止内容

- [ ] 未表示功效、安全性的断言或保证
- [ ] 未利用科研单位、学术机构、技术推广机构、行业协会或专业人士、用户的名义或形象作推荐、证明
- [ ] 未说明有效率
- [ ] 未违反安全使用规程
- [ ] 未贬低同类产品或与其他农药进行功效和安全性对比
- [ ] 未含评比、排序、推荐、指定、选用、获奖等综合性评价内容
- [ ] 未滥用未经国家认可的研究成果或不科学的词句、术语
- [ ] 未含"无效退款""保险公司保险"等承诺

## 必须标明

- [ ] 《农药广告审查批准文号》列为广告内容同时发布
