# 公益广告与其他特殊规定

## 公益广告专项规范

- [ ] 价值导向正确，符合国家法律法规和社会主义道德规范要求
- [ ] 体现国家和社会公共利益
- [ ] 语言文字使用规范
- [ ] 艺术表现形式得当，文化品位良好
- [ ] 公益广告内容与商业广告内容相区别
- [ ] 企业参与公益广告时，未标注商品或服务的名称以及其他与宣传、推销商品或服务有关的内容
- [ ] 未标注单位地址、网址、电话号码、其他联系方式等
- [ ] 平面作品标注企业名称和商标标识的面积未超过广告面积的1/5
- [ ] 音频、视频作品显示企业名称和商标标识的时间未超过5秒或总时长的1/5
- [ ] 未以公益广告名义变相设计、制作、发布商业广告

## 铁路站车广告特殊规范

- [ ] 依法取得《广告经营执照》
- [ ] 发布站车广告内容必须真实、健康，不得以任何形式欺骗用户和消费者
- [ ] 严格遵守国家的法律、法规和宣传政策
- [ ] 在铁路站车发布外商广告，经有关部门批准，并严格执行有关涉外规定
- [ ] 未利用铁路客、货各种票据（站台票除外）发布广告
- [ ] 未在客、货车厢外部及客车方向标志牌发布广告
- [ ] 未在站车设施上直接涂写（站区围墙、栅栏除外）
- [ ] 未在当地政府禁止设置广告的区域设置广告
- [ ] 未在客、货列车车次、名称上发布广告
- [ ] 未在影响行车安全、运输组织、环境保护的区域或形式设置广告
- [ ] 广告制作和设置突出"简洁、美观、安全"的原则，与站容车貌及服务工作相协调

## 军队武警名义使用限制

- [ ] 企业名称中未带有军队、武警部队番号、代号
- [ ] 企业名称中未含有"国防"、"八一"等特定含义字样
- [ ] 商品（服务）商标未冒用军队、武警部队、国防、八一等特定字样
- [ ] 未使用军旗、军（警）徽等标志
- [ ] 严禁以军队、武警部队名义发布商业广告
- [ ] 商业广告中未出现军用番号、代号、字样和标志图案
- [ ] 未使用"军品专营"、"军品代销"等用语
- [ ] 严禁在民用商品上印刷、刻制、铸造军旗、军（警）徽、八一、勋章等军用标志物
- [ ] 非军工定点企业未擅自制造军用标志物
- [ ] 军、警服装、鞋、帽、被装等产品作为民品上市销售的，未使用军、警图文标志
- [ ] 沿街店铺未打出"军需品"、"军用品"及类似招牌

## 出口广告管理规范

- [ ] 设立经营或兼营出口广告业务的企业经对外经济贸易部批准
- [ ] 取得批准后向工商行政管理机关办理注册登记手续
- [ ] 加入中国对外经济贸易广告协会
- [ ] 无出口广告业务经营权的企业委托境内取得出口广告经营权的企业代理
- [ ] 境内各类出口广告媒介通过出口广告经营单位征集广告
- [ ] 境外和港澳台地区广告商或媒介征集出口广告，经由境内有出口广告经营权的单位代理
- [ ] 年度出口广告专用外汇额度不少于年出口收汇额的千分之三
- [ ] 外商投资企业经营出口广告业务，经对外经济贸易部审批
