# 食品广告

## 一般食品

- [ ] 未涉及疾病预防、治疗功能
- [ ] 未宣传保健作用（一般食品）
- [ ] 未替代母乳
- [ ] 未涉及民族、风俗歧视

## 酒类

- [ ] 未诱导、怂恿饮酒
- [ ] 未展示饮酒动作
- [ ] 未与驾驶、操作绑定
- [ ] 未宣称缓解焦虑、增强体力等功效
- [ ] 未面向未成年人

## 进口食品

- [ ] 中文标签齐全

## 保健食品广告

### 审查与资质

- [ ] 未经审查不得发布保健食品广告
- [ ] 已取得《保健食品广告批准文号》并显著标明
- [ ] 产品已按规定注册或备案

### 内容红线

- [ ] 未涉及疾病预防、治疗功能
- [ ] 内容未超出注册证书或备案凭证、产品说明书范围
- [ ] 未使用科研单位、学术机构、行业协会或专家、学者、患者等名义或形象作推荐、证明
- [ ] 未利用医疗机构、医疗咨询、义诊、特约门诊等医疗服务内容
- [ ] 未含有"安全""安全无毒副作用""毒副作用小"等安全性保证内容
- [ ] 未含有"热销、抢购""家庭必备""免费赠送""评比、推荐""无效退款"等诱导性内容

### 必须标明

- [ ] 显著标明"**保健食品不是药物，不能代替药物治疗疾病**"
- [ ] 标明保健食品标志
- [ ] 标明适宜人群和不适宜人群

### 禁止宣传方式

- [ ] 未利用会议、讲座、健康咨询等方式进行虚假宣传

## 特殊医学用途配方食品广告

### 审查要求

- [ ] 未经审查不得发布广告
- [ ] 已取得《特殊医学用途配方食品广告批准文号》并显著标明

### 内容要求

- [ ] 内容未超出注册证书、产品标签、说明书范围
- [ ] 显著标明适用人群
- [ ] 显著标明"**不适用于非目标人群使用**"
- [ ] 显著标明"**请在医生或者临床营养师指导下使用**"

### 发布限制

- [ ] 特定全营养配方食品广告未在大众传播媒介或公共场所发布
- [ ] 未利用产品名称为各种活动冠名进行广告宣传

## 婴幼儿配方食品

- [ ] 未以选择性添加物质命名（添加食品安全国家标准规定的选择性添加物质的婴幼儿配方食品）

## 转基因食品

- [ ] 转基因食品已按规定显著标示
- [ ] 广告中未隐瞒或虚假宣传转基因属性

## 辐照食品

- [ ] 经辐照处理的食品已在食品名称临近位置标明"**辐照加工食品**"或"**经辐照处理**"
- [ ] 经辐照处理的配料已在配料表中注明

## 特殊食品标签一致性

- [ ] 特殊食品（保健食品、特殊医学用途配方食品、婴幼儿配方食品）的标签、说明书内容与注册或备案内容一致
