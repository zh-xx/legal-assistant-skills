# 不正当竞争

## 混淆行为

- [ ] 未擅自使用与他人有一定影响的商品名称、包装、装潢等相同或者近似的标识
- [ ] 未擅自使用他人有一定影响的名称（包括简称、字号等）
- [ ] 未擅自使用他人有一定影响的姓名（包括笔名、艺名、网名、译名等）
- [ ] 未擅自使用他人有一定影响的域名主体部分、网站名称、网页、新媒体账号名称、应用程序名称或者图标等
- [ ] 未擅自将他人注册商标、未注册的驰名商标作为企业名称中的字号使用
- [ ] 未将他人商品名称、企业名称（包括简称、字号等）、注册商标、未注册的驰名商标等设置为搜索关键词
- [ ] 未帮助他人实施混淆行为

## 虚假商业宣传

- [ ] 未对其商品的性能、功能、质量、销售状况、用户评价、曾获荣誉等作虚假或者引人误解的商业宣传
- [ ] 未通过组织虚假交易、虚假评价等方式帮助其他经营者进行虚假或者引人误解的商业宣传

## 商业秘密保护

- [ ] 未以盗窃、贿赂、欺诈、胁迫、电子侵入或者其他不正当手段获取权利人的商业秘密
- [ ] 未披露、使用或者允许他人使用以前项手段获取的权利人的商业秘密
- [ ] 未违反保密义务或者违反权利人有关保守商业秘密的要求，披露、使用或者允许他人使用其所掌握的商业秘密

## 损害商誉

- [ ] 未编造、传播或者指使他人编造、传播虚假信息或者误导性信息，损害其他经营者的商业信誉、商品声誉

## 网络不正当竞争

- [ ] 未在未经其他经营者同意的情况下，在其合法提供的网络产品或者服务中插入链接、强制进行目标跳转
- [ ] 未误导、欺骗、强迫用户修改、关闭、卸载其他经营者合法提供的网络产品或者服务
- [ ] 未恶意对其他经营者合法提供的网络产品或者服务实施不兼容
- [ ] 未以欺诈、胁迫、避开或者破坏技术管理措施等不正当方式，获取、使用其他经营者合法持有的数据

## 平台经营者义务

- [ ] 未强制或者变相强制平台内经营者按照其定价规则，以低于成本的价格销售商品
- [ ] 在平台服务协议和交易规则中明确平台内公平竞争规则
- [ ] 建立不正当竞争举报投诉和纠纷处置机制
- [ ] 发现平台内经营者实施不正当竞争行为的，及时依法采取必要的处置措施
- [ ] 保存有关记录，并按规定向平台经营者住所地县级以上人民政府监督检查部门报告

## 大型企业对中小企业的义务

- [ ] 未滥用自身资金、技术、交易渠道、行业影响力等方面的优势地位
- [ ] 未要求中小企业接受明显不合理的付款期限、方式、条件和违约责任等交易条件
- [ ] 未拖欠中小企业的货物、工程、服务等账款
