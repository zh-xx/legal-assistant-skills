# 兽药广告

## 禁止发布广告的兽药

- [ ] 兽用麻醉药品、精神药品以及兽医医疗单位配制的兽药制剂
- [ ] 所含成分的种类、含量、名称与兽药国家标准不符的兽药
- [ ] 临床应用发现超出规定毒副作用的兽药
- [ ] 国务院农牧行政管理部门明令禁止使用的，未取得兽药产品批准文号或未取得《进口兽药注册证书》的兽药

## 禁止内容

- [ ] 未表示功效、安全性的断言或保证
- [ ] 未利用科研单位、学术机构、技术推广机构、行业协会或专业人士、用户的名义或形象作推荐、证明
- [ ] 未说明有效率
- [ ] 未违反安全使用规程
- [ ] 未贬低同类产品或与其他兽药进行功效和安全性对比
- [ ] 未含"最高技术""最高科学""最进步制法""包治百病"等绝对化表示
- [ ] 未含评比、排序、推荐、指定、选用、获奖等综合性评价内容
- [ ] 未直接显示疾病症状和病理的画面
- [ ] 未含"无效退款""保险公司保险"等承诺

## 必须标明

- [ ] 《兽药广告审查批准文号》列为广告内容同时发布
