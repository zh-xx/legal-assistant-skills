# 广告行为与管理责任

## 1. 行政许可与资质

- [ ] 属于需行政审查广告（医药、医疗器械、保健食品等）
- [ ] 已完成事前审查
- [ ] 广告经营者、发布者具备合法资质

## 2. 广告管理

- [ ] 已签署书面广告合同
- [ ] 已留存广告内容及修改记录
- [ ] 已及时通知修改或撤稿

## 3. 户外广告

- [ ] 符合当地户外广告规范
- [ ] 已完成行政审批
- [ ] 未设置在禁止区域
- [ ] 定期维护并及时拆除

## 4. 广告档案

- [ ] 互联网广告档案留存：自广告发布行为终了之日起不少于 3 年
- [ ] 有奖销售档案留存：不少于 2 年

## 5. 个人信息保护

- [ ] 明示信息收集目的、方式、范围
- [ ] 已取得用户同意
- [ ] 未泄露或出售个人信息
- [ ] 未在未经同意的情况下发送商业信息
