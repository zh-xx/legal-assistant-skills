# 医疗器械广告

## 审查与资质

- [ ] 已取得《医疗器械广告证明》
- [ ] 产品已取得生产或经营准许证

## 禁止发布广告的医疗器械

- [ ] 未经批准生产的医疗器械
- [ ] 临床试用、试生产的医疗器械
- [ ] 未取得生产许可证生产的医疗器械（已实施生产许可证的产品）

## 禁止内容

- [ ] 未使用专家、医生、患者、未成年人或医疗科研、学术机构、医疗单位的名义进行宣传
- [ ] 未使用"保证治愈"等保证性断语
- [ ] 未与同类产品功效、性能进行比较
- [ ] 未运用数字或图表宣传治疗效果
- [ ] 未宣传不使用产品可能导致或加重某种疾病
- [ ] 未使用"第一""首创"等绝对性语言（无国家证明）

## 忠告语要求

- [ ] 推荐给个人使用的治疗性器械已标明"**请在医生指导下使用**"
